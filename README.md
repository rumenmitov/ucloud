## Setup
### Prerequisites
- [x] Node v18+ (https://nodejs.org/en/download/)
- [x] npm v9+
  
### Installation
1. Clone the program    
2. Install dependencies   
3. Start the program  
  
You can do so by following this script:
```  
git clone https://github.com/rumenmitov/ucloud.git   
npm install  
npm run start  
```
